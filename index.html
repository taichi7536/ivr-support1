支払い済み連絡受付サポートページ（試作品）

支払期日を過ぎた電気料金につきましては、お支払いをしていただいた上で、需給契約解約日の前営業日15時までに当社までご連絡をしていただく必要がございます。 このページでは、高圧電力をご契約のお客さま向けに、電気料金の支払い済み連絡をIVR（自動音声システム）で行う手順をご説明します。

⚠ 重要：音声案内の一部はスキップできません。指示に従って操作してください。

IVR操作手順

1. 専用の電話番号におかけください
2. 冒頭のアナウンスを最後までお聞きください
3. 「5」 を押してください
4. 「1」 を押して、支払済み連絡受付システムに入ります
5. 音声案内のガイダンスが続きますので、しばらくお待ちください

必要情報の入力手順

📌 ご契約者の確認
ご契約者本人の場合 → 「1」
それ以外の場合 → 「8」
📌 店所番号または事業所コードの入力（3桁 + #）
⚠ 領収書等で正しい番号を確認してください。
📌 お客さま番号の入力（13桁 + #）
📌 支払月分の入力（2桁 + #）
⚠ 例：1月分なら「01」
📌 お支払をした日付の入力（月2桁、日2桁 + #）
⚠ 例：3月6日なら「0306」
📌 お支払先の選択
銀行の場合 → 「1」
郵便局の場合 → 「2」
📌 お電話番号の入力（電話番号 + #）

受付完了の確認

⚠ 「電気料金支払い済みのお申し込みを受付しました」の音声を最後までお聞きください。

よくあるミスと対策

  • ✅ 店所番号が分からない → 領収書や請求書に表示されている3桁の番号です
  • ✅ 支払月分を間違えた → 領収書等で実際に支払った月分を確認してください
  • ✅ 申し込み完了前に電話を切った → 必ず最後まで音声を聞いてください
